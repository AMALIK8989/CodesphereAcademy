Python Syntax Basics

Overview

Python is a high-level programming language known for its simplicity and readability. It uses indentation to define code blocks instead of braces or keywords like other programming languages.


print("Hello, World!")

Variables and Data Types

In Python, you can declare variables without specifying their data types. Python supports various data types such as integers, floats, strings, lists, tuples, dictionaries, etc.


# Variable declaration
x = 10
name = "John"

Control Flow

Python provides constructs like if-else statements, loops (for and while), and exception handling for control flow.


# If-else statement
x = 10
if x > 0:
    print("Positive")
else:
    print("Negative")

Functions

Functions in Python are defined using the def keyword. They can accept parameters and return values.


# Function definition
def greet(name):
    return "Hello, " + name + "!"

# Function call
message = greet("Alice")
print(message)
